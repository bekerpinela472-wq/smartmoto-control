# Changelog

Todos los cambios importantes de este proyecto serán documentados en este archivo.

El formato está basado en Keep a Changelog y este proyecto utiliza Semantic Versioning (SemVer).

## [1.0.0] - 2026-06-01

### Added

* Creación inicial del proyecto SmartMoto Control.
* Implementación de la estructura base del repositorio.
* Desarrollo del módulo de gestión de motocicletas.
* Configuración inicial de la API REST.
* Documentación técnica del proyecto.
* Creación del tablero Scrum en Jira.
* Configuración del espacio colaborativo en Confluence.
* Documentación de endpoints mediante OpenAPI y Swagger.
* Publicación de la primera versión estable (Release v1.0.0).
