## 🚀 Descripción del Proyecto

SmartMoto Control es una aplicación web desarrollada con el propósito de facilitar la gestión de motocicletas mediante una plataforma digital intuitiva. El sistema permite almacenar información de los vehículos, consultar registros y administrar datos relacionados con su mantenimiento.

---

## ✨ Características

* Registro de motocicletas.
* Consulta de motocicletas registradas.
* Gestión centralizada de información.
* API REST documentada con OpenAPI.
* Control de versiones mediante Git y GitHub.
* Gestión de tareas utilizando Jira.
* Documentación colaborativa mediante Confluence.

---

## 🛠️ Tecnologías Utilizadas

* GitHub
* Jira Software
* Confluence
* Swagger / OpenAPI

## ⚙️ Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/usuario/smartmoto-control.git
```

### 2. Ingresar al proyecto

```bash
cd smartmoto-control
```

### 3. Instalar dependencias

```bash
npm install
```

### 4. Ejecutar la aplicación

```bash
npm run dev
```

---

## ▶️ Uso

Una vez iniciada la aplicación:

1. Acceder desde el navegador.
2. Registrar una motocicleta.
3. Consultar la información almacenada.
4. Gestionar los registros del sistema.
